# 🆓 FREE AI Setup - No Payment Required!

## Get Your FREE Hugging Face API Key

### Step 1: Create Account (FREE)
1. Go to: https://huggingface.co/join
2. Sign up with email (completely free)
3. Verify your email

### Step 2: Get API Token (FREE)
1. Go to: https://huggingface.co/settings/tokens
2. Click "New token"
3. Name it: "Doctor Know"
4. Type: "Read"
5. Click "Generate"
6. **Copy the token** (starts with `hf_...`)

### Step 3: Add Token to server.js
Open `server.js` and replace line 13:
```javascript
const HF_API_KEY = 'hf_your_token_here';
```

### Step 4: Install & Run
```bash
npm install
npm start
```
